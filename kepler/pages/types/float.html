Types/Float

Floats represent any decimal number, positive or negative.
When you require decimal precision, a Float is the best choice over an integer.

The following is an example of creating a variable with a Float type: 1 myFloat is 136.125

Casting Table

This table shows the types that can cast to a Float.
Type Castable
Bool Yes
Int Yes
uInt Yes
String Yes, if the value of the string is a valid number.
NaN No.
