Generic/Operations

Various operations in Kepler will result in a Boolean value.

Equality

"Equals" is a non-strict comparison of two values. Non-strict means that if the values types are not the same type, they will be converted to the same type and then compared. equals.kep 1 print 99 equals 100.0 2 3 print 25 equals 25.0 output False
True

Strict Equality

The only strict equality in Kepler is the "==" operator. This returns False if the values are not the same type. equals.kep 1 print 12 == 12.0 2 3 print 73 == 73 output False
True
