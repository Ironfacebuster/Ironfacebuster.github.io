Generic/Operations

Kepler has various operations to compare values, and control code flow.
Operations in Kepler will result in a Boolean value.

Equality

"Equals" is a non-strict comparison of two values. Non-strict means that if the values types are not the same type, they will be converted to the same type and then compared. equals.kep 1 print 99 equals 100.0 2 3 print 25 equals 25.0 output False
True

Strict Equality

The only strict equality in Kepler is the "==" operator. This returns False if the values are not the same type. equals.kep 1 print 12 == 12.0 2 3 print 73 == 73 output False
True

Greater Than

Greater than is a non-strict comparison. This checks if value "a" is greater than value "b". equals.kep 1 print 99 > 2.0 2 3 print u4 > 1003 output True
False

Greater Than or Equal To

Greater than or equal to is a non-strict comparison. This checks if value "a" is greater than, or equal to, the value of "b". equals.kep 1 print 56 >= 56.0 2 3 print 3 >= 50 output True
False

Less Than

Less than is a non-strict comparison. This checks if value "a" is less than value "b". equals.kep 1 print 99 < 2.0 2 3 print u4 < 1003 output True
False

Less Than or Equal To

Less than or equal to is a non-strict comparison. This checks if value "a" is less than, or equal to, the value of "b". equals.kep 1 print 35 <= 35.0 2 3 print 88 <= 23 output True
False
