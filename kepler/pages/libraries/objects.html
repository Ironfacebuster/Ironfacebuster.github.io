objects

Kepler has no built-in support for object oriented programming. However, it's possible to include the objects library to emulate some of the functionality.

create_obeject

Creates a new object.

Required Parameters

None.

Returns

(string) The object's ID.

object_delete

Delete an object.

Required Parameters

Name Type Description
id String The ID of the object.

Returns

Nothing.

object_exists

Check if an object exists.

Required Parameters

Name Type Description
id String The ID of the object.

Returns

(Boolean) True if the object exists, false otherwise.

object_get

Get a key/value pair from an object.

Required Parameters

Name Type Description
id String The ID of the object.
key String The key of the data you want to get.

Returns

(Any) The data associated with the key.

object_set

Set a key/value pair for an object.

Required Parameters

Name Type Description
id String The ID of the object.
key String The key of the data you want to set.
value Any The data to associate with the key.

Returns

Nothing.

object_set

Get a list of the keys an object has.

Required Parameters

Name Type Description
id String The ID of the object.

Returns

(String) The keys of the object, separated by commas.
